---
description: Review and promote pending reference file additions
---

Read `/reference/pending-review.md`.

If the file is empty (no entries below the marker line), say so and stop.

For each pending entry:

1. Show the entry clearly:
   - Target file
   - Source attribution
   - Date
   - Proposed content

2. Cross-check the proposed content:
   - Read the target reference file to check for duplicates or conflicts
   - If the entry contradicts existing content, flag the conflict
   - If the entry is a duplicate, flag it

3. Ask the student: **Promote, Edit, or Discard?**
   - **Promote** — append the content to the target reference file exactly as written
   - **Edit** — student provides corrections, then promote the edited version
   - **Discard** — delete the entry from pending-review.md

4. After processing each entry, remove it from `/reference/pending-review.md`

Process entries one at a time. Do not batch.
